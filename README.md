## ğŸ¯ Interactive Components

### 1. MouseFollower
Custom animated cursor that replaces the default browser cursor with a gold dot, outline ring, and click ripples.

### 2. InteractiveCards
Six principle cards with real-time 3D rotation based on mouse position, creating an immersive depth effect.

### 3. FloatingMenu
Floating action button with slide-out navigation menu and scroll progress bar at the top of the viewport.

### 4. GunAnimation
Interactive firearms training simulation with live statistics, auto-fire mode, and React state management.

### 5. ParticlesBackground
Canvas-based particle system creating a dynamic animated background.
